Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document (like an element, attribute, or text). This allows programming languages like JavaScript to interact with and manipulate the content, structure, and style of the document dynamically.

How do you select an element from the DOM?

I can select an element from the DOM using various methods in JavaScript:


  • document.getElementById('id'): Selects an element by its unique ID.
  • document.getElementsByClassName('class'): Selects elements by their class name.
  • document.getElementsByTagName('tag'): Selects elements by their tag name.
  • document.querySelector('selector'): Selects the first element that matches a CSS selector.
  • document.querySelectorAll('selector'): Selects all elements that match a CSS selector.
    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of adding separate listeners to individual child elements. When an event is triggered on a child element, it "bubbles" up the DOM to the parent, which can handle the event. This is useful because it reduces the number of event listeners needed, improves performance, and ensures that dynamically added elements can still respond to events without needing to attach new listeners.

    How do you manipulate an element's attributes and styles using the DOM?

    I can manipulate an element's attributes and styles in the DOM using JavaScript:


    Attributes:

  • element.setAttribute('attribute', 'value'): Sets a new attribute or changes the value of an existing attribute.
  • element.getAttribute('attribute'): Retrieves the value of an attribute.
  • element.removeAttribute('attribute'): Removes an attribute.

    • Styles:

  • element.style.property = 'value': Directly sets an inline style property (e.g., element.style.color = 'red').
  • element.classList.add('class'): Adds a class to an element.
  • element.classList.remove('class'): Removes a class from an element. element.
  • classList.toggle('class'): Toggles a class on or off based on its current state.
